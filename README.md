## Requirements
* Node.js >= 8.1

## Install
* Install Node.js and Git
* Open terminal:
```bash
$ git clone https://github.com/dsdenes/html-scramber
$ cd html-scramber
$ npm install
```
* Modify redirect.js file
* Copy your webpage into src/ folder
* Start obfuscation:
```bash
$ npm start
```

### Source html file path
Defaults to 'src/index.html'.
